Get Started

Built with Docker

Inception is a project where you need to create a fullstack website using Docker and only from hand using Dockerfile. For this project, i entirely use Alpine Linux on it version 3.19.1 for all my Dockerfile. You can find just under this scope all the container that i've created for this exercice.

Nginx + Mkcert

Nginx 1.20.1 was installed from source and added to the compilation the module redis to handle multi-site caching. The worst part of this project is the nginx config file that is the worst nightmare i had in a long time !!! For mkcert, pretty simple, it's just a package that automate the creation and the installation on a SSL/TLS certificate to secure test and dev environnement.

Mariadb

Mariadb installed form the dpkgs manager of Alpine Linux. Pretty easy part, a lot of documentation and AI model pretty well trained to help you for the configuration.

WordPress + Php83-fpm + Wp-cli

The 30% of my nightmare ! But the best thing in the world when everything show up and you only need to configure it using wp-cli.

Sftp (Bonus)

As asked in the bonus part, i added a docker with sftp installed on it with ssh key generated when you build the project. The SFTP point to the Wordpress volumes with the good right to you can edit, add or remove any file that you want !

Adminer (Bonus)

I think it's the easiest bonus, i've only created a docker, installed php83 and wget to download the php file. That's it ! Just to add a location inside the nginx config and you are good to go !

Redis Cache (Bonus)

For this bonus, you need to install redis on a docker, configure it to listen to cache request from nginx or php and that it ! I personally installed redis as a module of nginx and to achieve that, i needed to build from source nginx with the redis module ! Next, i just installed on wordpress a plugin that handle everything between php - wordpress and redis.

The static page (Bonus)!

Built with NodeJS

Last but not least, the static page bonus part ! As we can't use php, i decide to use nodejs because of a good previous experience with it. And thanks to a bootstrap template, you can stand on this beautiful nodejs + bootstrap template. i've just followed the installation step from a digital ocean post and voila !
