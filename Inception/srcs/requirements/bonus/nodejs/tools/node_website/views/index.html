Built with Docker

Inception is a project where you need to create a fullstack website using Docker and only from hand using Dockerfile. For this project, i entirely use Alpine Linux on it version 3.19.1 for all my Dockerfile. You can find just under this scope all the container that i've created for this exercice.

Nginx 1.20.1 + Redis (Bonus)

Nginx 1.20.1 was installed from source and added to the compilation the module redis to handle multi-site caching.The worst part of this projec tis the nginx configuration that is the worst nightmare i think i will never have in my dev life !

MariaDB + + Adminer

Mariadb installed form the dpkgs manager of Alpine Linux. Pretty easy part, a lot of documentation and AI model pretty well trained to help you for the configuration.

WordPress + Php83-fpm + Wp-cli + Redis (Bonus)

The half part of my nightmare ! But the best thing in the world when everything show up and you only need to configure it using wp-cli.

Sftp (Bonus)

As asked in the subject, i added a docker with sftp installed on it with ssh key generated when you make the project. The SFTP point to the Wordpress volumes with the good right to you can edit, add or remove any file that you want !
